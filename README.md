# AUTOPILOT PONG

Retro-futurist Pong with AI battle modes.

## Run

```bash
cd src
python3 game.py
```

## Controls

| Key | Action |
|-----|--------|
| W / UP | Move paddle up |
| S / DOWN | Move paddle down |
| SPACE | Start / Pause |
| 1 / 2 / 3 | AI difficulty |
| ESC | Quit |

## Modes

1. **VS AI** (default) - Play against AI
2. **ZEN** - Watch AI battle itself
3. **SYMBIOSIS** - AI assists your paddle

## Features

- CRT scanline effect
- Ball trail physics
- Screen shake on goals
- AI difficulty scaling
- Score tracking (first to 11)
